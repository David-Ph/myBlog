# myBlog

A blog app to share your mind.
A simple side project I created to try out EJS and QuillJS

## Technologies

- Javascript
- Node JS
- Express JS
- MongoDB
- Quill JS
- Bulma
- EJS

- [Creating a post with a rich text editor](https://github.com/David-Ph/good-omens-coffee#user-can-make-an-order)
